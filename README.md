# PhD Guide for Stockholm Students

This repository contains a LaTeX project for a comprehensive PhD guide, specifically tailored for PhD students at KTH Royal Institute of Technology and generally for those living in Stockholm.

## Topics Covered
- **Working Environment & Employment:** Admission, contracts, salary reviews, sick leave, and parental leave.
- **The PhD Program (Academic):** ISPs, Ladok, course registration, and dissertation guidelines.
- **Practical Information:** Housing, healthcare, public transport, and the local "fika" culture.
- **International Student Support:** Personal ID numbers, banking, and Swedish residence permits.
- **Research Ethics & Authorship:** Principles of good research and guidelines for human studies.

## Project Structure
- `main.tex`: The primary LaTeX document.
- `Solid Mechanics.tex`: Department-specific information for Solid Mechanics.
- `Fluids.tex`: Department-specific information for Fluid Mechanics (TBD).
- `Lightweight Structures.tex`: Department-specific information for Lightweight Structures (TBD).
- `.vscode/settings.json`: Pre-configured VS Code settings for LaTeX Workshop.

## How to Compile
You can compile this project using any LaTeX engine (like `pdfLaTeX` or `XeLaTeX`). If you are using VS Code, the project is configured to use the `LaTeX Workshop` extension.
